# csvstripper

## What it does
Python web app (Dockerized) that strips or filters CSV files. Tested with alarm log data.

## Stack
- Python
- Docker (Dockerfile present)

## Key files
- `app.py` — main application
- `dockerfile` — container setup
- `requirements.txt` — dependencies
- `alarm_log.csv` — test/example data
